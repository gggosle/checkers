# Technology Stack

## Languages & Runtime

- **JavaScript** (ES6+) - Primary language
- **HTML5** - Markup
- **CSS3** - Styling with CSS custom properties

## Architecture Pattern

- **MVC (Model-View-Controller)** - Clean separation of concerns
  - Models: Game logic and state management
  - Views: DOM rendering and user interface
  - Controllers: User input handling and coordination

## Key Dependencies

No external npm dependencies. Pure vanilla JavaScript with ES6 modules.

## Build/Development

- No build step required
- Native ES6 module imports via `<script type="module">`
- Direct browser execution

## Configuration

- `constants.js` - Central configuration for game rules and CSS classes
- `style.css` - CSS custom properties for theming
