# Checkers Game - Project Roadmap

## Project Overview
A browser-based checkers game with undo functionality.

## Requirements
- **UNDO-01**: Single undo button that undoes last move with backward animation
- **UNDO-02**: Button enables after first move, disables after undo
- **UNDO-03**: Prevents undo after game ends
- **UNDO-04**: Restores all game state (board, turn, mustJumpPiece, hasJumpsAvailable)
- **UNDO-05**: Handles multi-jump chains as single turn
